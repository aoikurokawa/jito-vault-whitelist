# Jito Vault Whitelist Program

## Instructions

- Initialize Config
- Initialize Whitelist
- Set Mint Burn Admin
- Set Merkle Root
- Mint
- Enqueue Withdrawal
- Burn Withdrawal Ticket
- Close Whitelist

## Setup

### Generate IDL and Clients

```bash
make generate-code
```

### Check Lint

```bash
make lint
```

### Test

```bash
make test
```
